### 파이썬 패키지: Pillow

```python
import matplotlib.pyplot as plt
import numpy as np
from PIL import Image

# 이미지 읽기
file_name = '../cv_images/tiger.jpg'
image = Image.open(file_name)

# 이미지 출력 : 새 윈도우에
image.show()

# 이미지 출력
plt.imshow(image)
plt.axis('off')
plt.show()

# 이미지 저장 : Image.save(File Path, format=None)
image.save("Test.bmp",'BMP')

# 이미지 데이터 배열로 저장하기
```
